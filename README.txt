Naval-Battle for Android

This is a project developed in the Tecnichal High School of Campinas(COTUCA) for students during their last year.
This is a game of Battle-Naval designed for Android devices. The game works using Bluetooth comunication and the board is proportional with the device's screen.
As soon possible, more information will be avaliable.

You can send sugestions, critics or opinion to lucianosabenca@gmail.com


Batalha Naval Android

Este projeto foi desenvolvido por alunas do Colégio Técnico de Campinas(Cotuca) durante o 3º ano do Ensino médio.
Este projeto é um jogo de Batalha Naval, desenhado para dispositvos Android. O jogo usa comunicação Bluetooth e o tabuleiro é proporcional à tela do dispositivo.
Assim que possivel, mais informaçoes estarão diponiveis.
